Generate examples for [mutation testing] of public APIs by [OpenAPI 2.0 schema object] descriptions.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'locust'
```

And then execute:

```shell
$ bundle
```

Or install it yourself as:

```shell
$ gem install locust
```
